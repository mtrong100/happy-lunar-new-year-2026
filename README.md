# Lunar New Year Countdown 2026

Welcome to the Lunar New Year Countdown 2026 website! This project is a countdown timer for the upcoming Lunar New Year in 2026, featuring a fireworks effect.

## Screenshot

![Screenshot](./Screenshot.png)

## Installation

1. Clone the repository
2. Install dependencies:
   - Run `npm install` to install the required dependencies.
3. Start the application:
   - Run `npm start` to start the development server.

## Usage

1. Open your web browser and navigate to `http://localhost:3000`.
2. The countdown timer will automatically start counting down to the Lunar New Year in 2026.
3. Enjoy the fireworks effect as the countdown progresses!
